# AccessFlix

Hub de acessibilidade para filmes e jogos no Brasil. Ajuda pessoas com deficiência (PCD) a encontrar conteúdo com os recursos de acessibilidade que precisam — audiodescrição, legendas, Libras, linguagem simplificada, alto contraste e mais.

## Stack

- Vite + React 18 + TypeScript
- Tailwind CSS v4
- Radix UI / shadcn/ui
- Lucide React (ícones)
- Motion (animações)

## Rodando localmente

```bash
npm install
npm run dev
```

App em `http://localhost:5173`.

## Build

```bash
npm run build
```
